텍스트 CSS

font-family=글꼴

한글과 영문이 함께 작성되는 경우 영문폰트를 앞에 작성하고 한글폰트를 뒤에 작성한다. 이유는 한글폰트는 영문폰트가 포함된 경우가 많지만 영문폰트는 한글이 포함되어 있지 않거나 기본폰트로 지정된다.

font-size=글자의 크기

px, %, em, rem, vw, vh, small, large ... 글자크기를 따로 지정하지 않은 경우 기본 글자 크기는 16px이며, 100%는 기본글자 크기를 기본으로 100% 적용된다.

font-weight=글자의 두께

100~900까지 수치값으로 조절이 가능하고, lighter(100)/ normal(400)/bold(700)/ bolder(900) 으로 사용가능하다. 글꼴마다 지원하는 두께가 다르다.

font-style=글자의 기울임

normal(기본값), italic(기울임)

인천시 남동구 구월동 신현프라자 9층 SBS아카데미컴퓨터아트학원

text-decoration=글자의 선

underline(밑줄) ,overline(윗줄) , line-through(삭선), none(제거)

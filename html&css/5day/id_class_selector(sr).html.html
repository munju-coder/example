id선택자와 class선택자

id & class 네이밍 규칙

  • 영문, 숫자, _, - 등을 이용하여 이름을 작성한다.(한글, 특수문자, 공백 등을 사용할수없다)
  • 첫글자는 반드시 영문자를 사용한다.
  • ID는 한 페이지 내에서 중복된 이름을 여러번 지정하지 않는다.
  • CLASS는 한 페이지 내에서 중복된 이름을 여러번 지정 가능하고, 하나의 태그에 여러개의 CLASS를 지정하는것도 가능하다.
  • 이름에 의미를 부여하여 이름을 보고 어떤 역할을 하는지 알수있도록 한다.
  • 태그이름, 속성이름, 속성값 등을 피해서 작성한다. (여러단어를 조합하거나 단어를 줄여서 작성하여 중복을 피한다)
  • 여러단어를 조합하는 경우 단어간 구분을 짓는다.

네이밍 규칙 예시

  • abc
  • abc1
  • 1abc
  • ab_c
  • ab-c
  • _abc

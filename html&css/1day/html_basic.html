strong태그와 b태그는 글자를 굵게 만들어 준다.

em태그와 i태그는 글자에 기울임을 만들어준다.

ins태그는 밑줄, del태그 삭선(취소선)을 만들어준다.

mark태그는 글자에 형광표시를 만들어준다.
